Chakra Community Repository Tools

The Chakra Community Repository tools are a collection of small apps and libraries to help users interact with the CCR. These tools include ccr, the CCR package installation script, ccr-tools, a small script for talking to CCR via CLI, meant mostly for package maintainers, python-ccr, a python library for interacting with the CCR, and aur2ccr, a program that automatically converts packages from AUR and the Arch repos to CCR source packages. You can install ccr and ccr-tools with pacman -S ccr, python-ccr with ccr -S python-ccr (NOTE: python-ccr will be added to the CCR when version 0.2 is complete. Until then, build it manually with the provided PKGBUILD.), and aur2ccr with ccr -S aur2ccr.

Example usage:

# Convert [package] from AUR or Arch repos to CCR package:
aur2ccr -m "My Name <email@example.com>" [package]
# Upload [package] to CCR:
ccr-tools --submit [package]/[package]/[package]-[pkgver]-[pkgrel].src.tar.gz
# Vote for [package] in CCR:
ccr-tools --vote [package]
# Install the package from CCR
ccr -S [package]



Searching:

# Search CCR for [package]:
ccr -Ss [package]
# Search AUR for [package]:
aur2ccr --search [package]
# Search for packages owned by [username] in CCR:
ccr-tools --maintainer [username]
# Searching with python-ccr:
>>> import ccr
>>> ccr.search('[package]')
>>> ccr.msearch('[username]')
